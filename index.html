projects ~ % what I've been working on!

projects ~ % cat ece_faculty_directory.txt

I built a full-stack web application for the Purdue ECE Faculty Directory to make finding faculty fast and easy. It features a clean React frontend with animations, backed by a Node and Express server that supports searching by name or research area. To improve search flexibility, I added wildcard, partial, and root-word matching using regex and porter stemming. I also built a web scraper that collects and caches faculty data so the app stays fast without making unnecessary requests. Check it out here!

projects ~ % cat task_tracker.txt

I built a personalized weekly task planner (heavily inspired by Notion) to help organize tasks by day of the week. It features a React and TypeScript frontend with animations and a drag-and-drop interface powered by the HTML5 Drag and Drop API. To improve the user's experience, I implemented scroll-based scaling animations where calendar boxes dynamically enlarge as you scroll, along with hover and click animations throughout the interface. The app allows full customization with multiple color themes (mine is set to pink of course), custom emoji selection that updates the favicon dynamically, and persistent data storage using localStorage. I also added an optional "other tasks" section separate from daily tasks, a weekend toggle, and inline task editing. Try it out here!
